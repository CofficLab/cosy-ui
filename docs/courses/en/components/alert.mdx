## Introduction

The Alert component is used to display important messages to users, supporting multiple types (info/success/warning/error) and custom styles.

Component features:
- Supports four preset types, each with corresponding icons and colors
- Can add titles to make alerts more prominent
- Supports custom classes to override default styles
- Responsive design, adapts to different screen sizes

## Basic Usage

The simplest way to display an alert message:

<CodeExample
  code={AlertExampleCodes.Basic}
>
  <AlertBasic />
</CodeExample>

## Alert with Title

Add a title using the title property:
