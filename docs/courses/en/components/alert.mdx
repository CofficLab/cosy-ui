import { Alert, AlertPackage } from '@cosy/index';

## Introduction

The Alert component is used to display important prompt information to users, supporting various types (info/success/warning/error) and custom styles.

Component features:
- Supports four preset types, each with corresponding icons and colors
- Can add titles to make prompts more prominent
- Supports custom classes to override default styles
- Responsive design, adapting to different screen sizes

## Basic Usage

The simplest way to display prompt information:

<AlertPackage.AlertContainers.Basic/>

## Alert with Title

Add a title using the title property:

<AlertPackage.AlertContainers.WithTitle/>

## Alert Types

The Alert component supports four preset types:

<AlertPackage.AlertContainers.Types/>

## Custom Styles

Customize the alert box style using the class property:

<AlertPackage.AlertContainers.CustomStyle/>
