## Basic Usage

The most basic product collection display, automatically adjusting the layout based on screen size

<CodeExample
  code={`<Products
  products={[
    {
      name: "Product 1",
      image: "/images/products/product1.jpg",
      description: "This is a powerful example product that provides a quality experience for users.",
      productUrl: "https://example.com/product1"
    },
    {
      name: "Product 2",
      image: "/images/products/product2.jpg",
      description: "Another excellent product with innovative features and an intuitive interface.",
      productUrl: "https://example.com/product2"
    }
  ]}
/>`}
>
  <Products
    products={[
      {
        name: "Product 1",
        image: getExampleImage({ width: 300, height: 200 }),
        description: "This is a powerful example product that provides a quality experience for users.",
        productUrl: "https://example.com/product1"
      },
      {
        name: "Product 2",
        image: getExampleImage({ width: 300, height: 200 }),
        description: "Another excellent product with innovative features and an intuitive interface.",
        productUrl: "https://example.com/product2"
      }
    ]}
  />
</CodeExample>

## Properties

| Property | Type | Default | Description |
| --- | --- | --- | --- |
| products | ProductItem[] | - | Array of product data, each item contains all properties of the ProductCard component (except class) |
| layout | 'grid' \| 'list' | 'grid' | Layout type, grid or list |
| columns | number \| ResponsiveColumns | \{ base: 1, sm: 2, md: 3, lg: 4 } | Column configuration, can be a fixed number or responsive object |
| cardSize | 'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' | 'md' | Card size, applied to all ProductCards |
| gap | 'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' | 'md' | Card spacing |
| equalHeight | boolean | true | Whether to enable equal height cards |
| descriptionLines | number | - | Maximum number of lines for description text |
| class | string | - | Custom class name |
