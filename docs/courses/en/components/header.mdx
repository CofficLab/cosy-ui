## Basic Header

The simplest way to use the Header component, displaying the website logo and navigation bar

<CodeExample
  code={`<Header 
  logo={{ src: "https://via.placeholder.com/100", width: 100, height: 100, format: "png" }}
  logoHref="/"
/>`}
>
  <Header 
    logo={{ src: "https://via.placeholder.com/100", width: 100, height: 100, format: "png" }}
    logoHref="/"
    sticky={false}
  />
</CodeExample>

## Header with Navigation Menu

Add navigation menu through the navItems property

<CodeExample
  code={`<Header 
  logo={{ src: "https://via.placeholder.com/100", width: 100, height: 100, format: "png" }}
  logoHref="/"
  navItems={[
    { href: "/", label: "Home", match: (path) => path === "/" },
    { href: "/products", label: "Products", match: (path) => path.startsWith("/products") },
    { href: "/about", label: "About Us", match: (path) => path.startsWith("/about") }
  ]}
/>`}
>
  <Header 
    logo={{ src: "https://via.placeholder.com/100", width: 100, height: 100, format: "png" }}
    logoHref="/"
    sticky={false}
    navItems={[
      { href: "/", label: "Home", match: (path) => path === "/" },
      { href: "/products", label: "Products", match: (path) => path.startsWith("/products") },
      { href: "/about", label: "About Us", match: (path) => path.startsWith("/about") }
    ]}
  />
</CodeExample>

## Header with Language Switching

Add language switching functionality through the languages and currentLocale properties

<CodeExample
  code={`<Header 
  logo={{ src: "https://via.placeholder.com/100", width: 100, height: 100, format: "png" }}
  logoHref="/"
  languages={['zh-cn', 'en']}
  currentLocale="en"
/>`}
>
  <Header 
    logo={{ src: "https://via.placeholder.com/100", width: 100, height: 100, format: "png" }}
    logoHref="/"
    sticky={false}
    languages={['zh-cn', 'en']}
    currentLocale="en"
  />
</CodeExample>

## Header with Base Path

Set the base path through the basePath property for websites deployed in subdirectories

<CodeExample
  code={`<Header 
  logo={{ src: "https://via.placeholder.com/100", width: 100, height: 100, format: "png" }}
  logoHref="/"
  basePath="/docs"
  languages={['zh-cn',  'en']}
  currentLocale="en"
/>`}
>
  <Header 
    logo={{ src: "https://via.placeholder.com/100", width: 100, height: 100, format: "png" }}
    logoHref="/"
    basePath="/docs"
    sticky={false}
    languages={['zh-cn', 'en']}
    currentLocale="en"
  />
</CodeExample>

## Header with Sidebar Toggle

Add a sidebar toggle button through the showSidebarToggle property, suitable for documentation websites

<CodeExample
  code={`<Header 
  logo={{ src: "https://via.placeholder.com/100", width: 100, height: 100, format: "png" }}
  logoHref="/"
  showSidebarToggle={true}
  defaultSidebarOpen={false}
/>`}
>
  <Header 
    logo={{ src: "https://via.placeholder.com/100", width: 100, height: 100, format: "png" }}
    logoHref="/"
    sticky={false}
    showSidebarToggle={true}
    defaultSidebarOpen={false}
  />
</CodeExample>
