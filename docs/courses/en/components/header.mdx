The Header component is a layout component for website header navigation, providing features like logo, navigation menu, and language switching.

The component supports responsive design, offering good display effects on different screen sizes, and can optionally be fixed at the top of the page.

This documentation only lists common usage examples. For more usage examples, please refer to the component's source code.

## Basic Usage

The basic Header component includes logo and language switching functionality:

<CodeExample
  code={HeaderExampleCodes.Basic}
>
  <HeaderBasic />
</CodeExample>

## Custom Height

Use the `height` property to set the navigation bar height. Seven preset sizes are supported:

<CodeExample
  code={`<Header 
  logo={{ src: getExampleImage({ width: 100, height: 100, provider: "robohash", randomSeed: "cosy" }) }}
  logoHref="/"
  height="sm"
/>`}
>
  <Header 
    logo={{ src: getExampleImage({ width: 100, height: 100, provider:"picsum", tag: "tech" }), width: 100, height: 100, format: "png" }}
    logoHref="/"
    sticky={false}
    height="sm"
    rounded="lg"
  />
</CodeExample>

Height comparison:

- `3xs`: Micro size (16px) - Suitable for minimalist interfaces or embedded components
- `2xs`: Extra small size (24px) - Suitable for compact toolbars
- `xs`: Extra small size (32px) - Suitable for auxiliary navigation bars
- `sm`: Small size (40px) - Suitable for lightweight tool websites
- `md`: Medium size (48px, default) - Suitable for most applications
- `lg`: Large size (64px) - Suitable for content-centric websites
- `xl`: Extra large size (80px) - Suitable for brand showcase websites

The height setting automatically adjusts the logo size and top page spacing (when `sticky` is true).

## Custom Position

Use the `position` property to set the navigation bar position:

<CodeExample
  code={HeaderExampleCodes.CustomPosition}
>
  <HeaderCustomPosition />
</CodeExample>

Position options:
- `start`: Left-aligned (default)
- `center`: Center-aligned
- `end`: Right-aligned

## Customizing the Navbar Content

The Header component provides three slots for customizing different parts of the navigation bar:

- `navbar-start`: Customize the left side of the navigation bar
- `navbar-center`: Customize the middle section of the navigation bar
- `navbar-end`: Customize the right side of the navigation bar (displays language switcher by default)

Here are examples of customizing each section:

### Customizing the Left Side

You can add custom content next to the logo:

>

</CodeExample>

### Customizing the Middle Section
