## Quick Start

Install via npm:

```bash
npm install @coffic/cosy-ui
```

Or using pnpm:

```bash
pnpm add @coffic/cosy-ui
```

<Alert type="warning">
    The current version is a Beta version, and there may be unstable situations, please use it with caution.
</Alert>
