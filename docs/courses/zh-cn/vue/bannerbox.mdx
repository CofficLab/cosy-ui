## 自定义背景

通过 `backgroundClassIndex` 属性选择预设的渐变背景。

<CodeExample
  code={BannerBoxExampleCodes.CustomBg}
>
  <BannerBoxExamples.CustomBg client:load />

## 下载按钮显示控制

通过 `displayMode` 属性控制下载按钮的显示方式，支持三种模式：`always`（总是显示）、`hover`（鼠标悬停时显示）和`never`（不显示）。

<CodeExample
  code={BannerBoxExampleCodes.DisplayMode}
>
  <BannerBoxExamples.DisplayMode client:load />
</CodeExample>

## 尺寸预设

组件内置多种尺寸预设，可通过下载按钮下拉菜单选择，组件会自动记住上次选择的尺寸。

<CodeExample
  code={BannerBoxExampleCodes.SizePreset}
>
  <BannerBoxExamples.SizePreset client:load />
</CodeExample>

## 导出为图片

BannerBox 组件支持将内容导出为图片，通过左上角的下载按钮操作。

<CodeExample
  code={BannerBoxExampleCodes.ImageExport}

通过传入 `title`、`description` 和 `features` 属性使用智能横幅模式，自动展示标题、描述和特性列表。

<CodeExample

BannerBox 支持集成自定义组件，可以在横幅中插入任意 Vue 组件。

## 属性

| 属性名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| displayMode | string | 'hover' | 下载按钮显示模式：'always'(总是显示),'hover'(悬停显示),'never'(不显示) |
| backgroundClassIndex | number | 0 | 背景样式索引，对应内置的背景样式列表 |
| title | string | '' | 横幅标题 |
| description | string | '' | 横幅描述 |
| features | Array | [] | 特性列表，每项包含 \{emoji, title, link\} |
| customComponent | Object | null | 自定义组件 |
| customComponentProps | Object | {} | 自定义组件的属性 |

## 插槽

| 插槽名 | 说明 |
| --- | --- |
| default | 横幅内容，当不使用 title/description/features 属性时显示 |
