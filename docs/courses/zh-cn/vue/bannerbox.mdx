最基本的横幅容器，可放置任何内容。

<CodeExample
  code={`<template>
  <BannerBox>

## 下载按钮显示控制

        <p>下载按钮始终可见</p>
      </div>
    </BannerBox>

    <h3 class="text-lg font-medium">悬停时显示下载按钮（默认）</h3>
    <BannerBox displayMode="hover">
      <div class="flex items-center justify-center min-h-[100px]">
        <p>鼠标悬停时显示下载按钮</p>
      </div>
    </BannerBox>

    <h3 class="text-lg font-medium">隐藏下载按钮</h3>
    <BannerBox displayMode="never">
      <div class="flex items-center justify-center min-h-[100px]">
        <p>不显示下载按钮</p>
      </div>
    </BannerBox>
  </div>
</template>

<script setup>
import { BannerBox } from 'cosy-ui';

## 自定义组件集成

BannerBox 支持集成自定义组件，可以在横幅中插入任意 Vue 组件。

<CodeExample
  code={`<template>
  <BannerBox
    title="带自定义组件的横幅"
    description="在横幅中集成其他UI组件"
    :features="[
      {
        emoji: '🎨',
        title: '可定制UI',
        link: '#customize'
      }
    ]"
    :customComponent="Button"
    :customComponentProps="{ 
      text: '立即注册', 
      variant: 'primary',
      size: 'lg',
      onClick: () => alert('注册按钮点击')
    }"
  />
</template>

<script setup>
import { BannerBox, Button } from 'cosy-ui';
</script>`}
>
  <p class="cosy:text-sm cosy:mb-4">此处为示例展示，实际使用时可以设置任意Vue组件</p>
</CodeExample>

## 属性

| 属性名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| displayMode | string | 'hover' | 下载按钮显示模式：'always'(总是显示),'hover'(悬停显示),'never'(不显示) |
| backgroundClassIndex | number | 0 | 背景样式索引，对应内置的背景样式列表 |
| title | string | '' | 横幅标题 |
| description | string | '' | 横幅描述 |
| features | Array | [] | 特性列表，每项包含 \{emoji, title, link\} |
| customComponent | Object | null | 自定义组件 |
| customComponentProps | Object | {} | 自定义组件的属性 |

## 插槽

| 插槽名 | 说明 |
| --- | --- |
| default | 横幅内容，当不使用 title/description/features 属性时显示 |
