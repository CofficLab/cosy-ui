## 基础用法

最基本的对话框展示方式。在实际应用中，您需要使用 `v-model` 来控制对话框的显示状态。

  code={AlertDialogExampleCodes.Basic}
>

</CodeExample>

## 多语言支持

通过自定义 `ok-text` 和 `cancel-text` 属性来支持不同语言。

<CodeExample
  code={AlertDialogExampleCodes.Multilang}
>
  <AlertDialogExamples.Multilang client:load />
</CodeExample>

## 属性

| 属性名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| modelValue | boolean | - | 控制对话框的显示状态 |
| message | string | - | 对话框显示的消息内容 |
| ok-text | string | '确定' | 确认按钮文本 |
| cancel-text | string | '取消' | 取消按钮文本 |
