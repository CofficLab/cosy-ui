## FeatureButton 属性

| 属性名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| title | string | 'Feature Button' | 按钮文本 |
| size | string | 'w-64' | 按钮大小，可选值：'w-64', 'w-32', 'w-16', 'w-12', 'w-8' |
| lang | string | 'en' | 语言设置，可选值：'en', 'zh' |
| showTips | boolean | false | 是否显示提示信息 |

## LinkButton 属性

| 属性名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| text | string | - | 按钮文本 |
| href | string | - | 链接地址 |
| openInNewTab | boolean | true | 是否在新标签页打开链接 |
