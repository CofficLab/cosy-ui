## 不显示边框

通过 `showFrame` 属性控制是否显示 iPhone 边框。

<CodeExample
  code={`<template>
  <iPhoneWindow :showFrame="false">
    <div class="p-4 text-center">
      <h2 class="text-xl font-bold mb-2">纯界面模式</h2>
      <p>不显示 iPhone 边框，只展示内容</p>
    </div>
  </iPhoneWindow>
</template>

<script setup>
import { iPhoneWindow } from 'cosy-ui';
</script>`}
>
  <iPhoneWindowExamples.NoFrame client:load />
</CodeExample>

## 属性

| 属性名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| height | string | 'h-96' | 窗口高度 |
| title | string | '' | 窗口标题 |
| statusBarButtons | array | [] | 状态栏按钮数组 |
| withShadow | boolean | true | 是否显示阴影效果 |
| showFrame | boolean | true | 是否显示 iPhone 边框 |
| backgroundColor | string | '' | 内容区域背景色 |

## 插槽

| 插槽名 | 说明 |
| --- | --- |
| default | 主要内容区域 |
