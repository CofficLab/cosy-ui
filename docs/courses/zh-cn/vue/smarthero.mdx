## 属性

| 属性名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| title | string | - | 标题文本 |
| description | string | - | 描述文本 |
| image | ImageProps | `{ src: '', alt: '' }` | 图片配置对象 |
| links | Link[] | `[]` | 链接数组 |

## 类型定义

```ts
interface Link {
    text: string;    // 链接文本
    href: string;    // 链接地址
}

interface ImageProps {
    src: string;     // 图片地址
    alt: string;     // 图片描述
}
```

## 插槽

| 插槽名 | 说明 |
| --- | --- |
| app | 用于在标题和链接之间插入自定义内容 |
