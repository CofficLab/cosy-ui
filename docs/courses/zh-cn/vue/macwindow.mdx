## 基础用法

最基本的窗口展示方式，带有标题和窗口控制按钮。

<CodeExample
  code={`<template>
  <MacWindow title="代码编辑器">
    <div class="p-4">
      窗口内容区域
    </div>
  </MacWindow>
</template>

<script setup>
import { MacWindow } from 'cosy-ui';
</script>`}
>
  <MacWindow title="代码编辑器">
    <div class="cosy:p-4">
      窗口内容区域
    </div>
  </MacWindow>
</CodeExample>

## 标签页

通过 `tabs` 属性和 `v-model` 创建带标签页的窗口。

<CodeExample
  code={`<template>
  <MacWindow
    title="设置"
    v-model="activeTab"
    :tabs="[
      { label: '通用', value: 'general' },
      { label: '外观', value: 'appearance' },
      { label: '高级', value: 'advanced' }
    ]"
  >
    <div class="p-4">
      <div v-if="activeTab === 'general'">通用设置内容</div>
      <div v-if="activeTab === 'appearance'">外观设置内容</div>
      <div v-if="activeTab === 'advanced'">高级设置内容</div>
    </div>
  </MacWindow>
</template>

<script setup>
import { ref } from 'vue';
import { MacWindow } from 'cosy-ui';

const activeTab = ref('general');
</script>`}
>
  <MacWindow
    title="设置"
    modelValue="general"
    tabs={[
      { label: '通用', value: 'general' },
      { label: '外观', value: 'appearance' },
      { label: '高级', value: 'advanced' }
    ]}
  >
    <div class="cosy:p-4">
      <div>通用设置内容（在实际使用中，可以根据选中的标签动态显示内容）</div>
    </div>
  </MacWindow>
</CodeExample>

## 工具栏和状态栏

使用 `toolbar` 和 `status` 插槽自定义工具栏和状态栏内容。

<CodeExample
  code={`<template>
  <MacWindow title="文件浏览器">
    <!-- 工具栏插槽 -->
    <template #toolbar>
      <button class="btn btn-ghost btn-sm" @click="handleSearch">
        <SearchIcon class="w-4 h-4" />
      </button>
      <button class="btn btn-ghost btn-sm" @click="handleSettings">
        <SettingsIcon class="w-4 h-4" />
      </button>
    </template>
    
    <!-- 主内容 -->
    <div class="p-4">
      窗口内容区域
    </div>
    
    <!-- 状态栏插槽 -->
    <template #status>
      <div class="text-xs">就绪</div>
      <button class="btn btn-ghost btn-xs" @click="handleInfo">
        <InfoIcon class="w-4 h-4" />
      </button>
    </template>
  </MacWindow>
</template>

<script setup>
import { MacWindow } from 'cosy-ui';
import { SearchIcon, SettingsIcon, InfoIcon } from 'cosy-ui/icons';

const handleSearch = () => {
  console.log('搜索');
};

const handleSettings = () => {
  console.log('设置');
};

const handleInfo = () => {
  console.log('信息');
};
</script>`}
>
  <MacWindow title="文件浏览器">
    <div slot="toolbar">
      <button class="cosy:btn cosy:btn-ghost cosy:btn-sm">
        <SearchIcon class="cosy:w-4 cosy:h-4" />
      </button>
      <button class="cosy:btn cosy:btn-ghost cosy:btn-sm">
        <SettingsIcon class="cosy:w-4 cosy:h-4" />
      </button>
    </div>
    
    <div class="cosy:p-4">
      窗口内容区域
    </div>
    
    <div slot="status">
      <div class="cosy:text-xs">就绪</div>
      <button class="cosy:btn cosy:btn-ghost cosy:btn-xs">
        <InfoIcon class="cosy:w-4 cosy:h-4" />
      </button>
    </div>
  </MacWindow>
</CodeExample>

## 属性

| 属性名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| height | string | 'h-96' | 窗口高度 |
| title | string | '' | 窗口标题 |
| withShadow | boolean | true | 是否显示阴影效果 |
| tabs | array | [] | 标签页数组，每个标签包含label和value属性 |
| modelValue | string | '' | 当前选中的标签页value值，可使用v-model绑定 |

## 插槽

| 插槽名 | 说明 |
| --- | --- |
| default | 窗口主要内容 |
| sidebar | 侧边栏内容 |
| toolbar | 工具栏内容，位于标题栏右侧 |
| status | 状态栏内容，位于窗口底部 |

## 事件

| 事件名 | 参数 | 说明 |
| --- | --- | --- |
| close | - | 点击关闭按钮时触发 |
| minimize | - | 点击最小化按钮时触发 |
| maximize | - | 点击最大化按钮时触发 |
| update:modelValue | value | 切换标签页时触发，用于v-model |
