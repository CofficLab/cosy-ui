## 标签页

传递字符串数组作为标签页。组件内部会自动管理当前激活的标签，点击标签会触发 `onTabClick` 回调，您需要自行维护标签状态。

<CodeExample
  code={`<template>
  <MacWindow
    title="设置"
    :tabs="['通用', '外观', '高级']"
    :onTabClick="handleTabClick"
  >
    <div class="p-4">
      <div v-if="activeTab === '通用'">通用设置内容</div>
      <div v-if="activeTab === '外观'">外观设置内容</div>
      <div v-if="activeTab === '高级'">高级设置内容</div>
    </div>
  </MacWindow>
</template>

<script setup>
import { ref } from 'vue';
import { MacWindow } from 'cosy-ui';

const activeTab = ref('通用');

const handleTabClick = (tab) => {
  activeTab.value = tab;
  console.log('当前标签:', tab);
};
</script>`}
>
  <MacWindowExamples.WithTabs client:load />
</CodeExample>

## 事件处理

通过 `onCloseWindow`、`onMinimizeWindow` 和 `onMaximizeWindow` 属性处理窗口控制按钮的点击事件。还可以使用 `onTabClick` 监听标签切换。

<CodeExample
  code={`<template>
  <MacWindow 
    title="应用窗口"
    :tabs="['信息', '设置', '帮助']"
    :onTabClick="handleTabClick"
    :onCloseWindow="handleClose"
    :onMinimizeWindow="handleMinimize"
    :onMaximizeWindow="handleMaximize"
  >
    <div class="p-4">
      <div v-if="activeTab === '信息'">信息内容</div>
      <div v-if="activeTab === '设置'">设置内容</div>
      <div v-if="activeTab === '帮助'">帮助内容</div>
    </div>
  </MacWindow>
</template>

<script setup>
import { ref } from 'vue';
import { MacWindow } from 'cosy-ui';

const activeTab = ref('信息');

const handleTabClick = (tab) => {
  activeTab.value = tab;
  console.log('切换到标签:', tab);
};

const handleClose = () => {
  alert('关闭窗口');
};

const handleMinimize = () => {
  alert('最小化窗口');
};

const handleMaximize = () => {
  alert('最大化窗口');
};
</script>`}
>
  <MacWindowExamples.WithEvents client:load />
</CodeExample>

## 工具栏和状态栏

使用 `toolbar` 和 `status` 插槽自定义工具栏和状态栏内容。

<CodeExample
  code={`<template>
  <MacWindow title="文件浏览器">
    <!-- 工具栏插槽 -->
    <template #toolbar>
      <button class="btn btn-ghost btn-sm" @click="handleSearch">
        <SearchIcon class="w-4 h-4" />
      </button>
      <button class="btn btn-ghost btn-sm" @click="handleSettings">
        <SettingsIcon class="w-4 h-4" />
      </button>
    </template>
    
    <!-- 主内容 -->
    <div class="p-4">
      窗口内容区域
    </div>
    
    <!-- 状态栏插槽 -->
    <template #status>
      <div class="text-xs">就绪</div>
      <button class="btn btn-ghost btn-xs" @click="handleInfo">
        <InfoIcon class="w-4 h-4" />
      </button>
    </template>
  </MacWindow>
</template>

<script setup>
import { MacWindow } from 'cosy-ui';
import { SearchIcon, SettingsIcon, InfoIcon } from 'cosy-ui/icons';

const handleSearch = () => {
  console.log('搜索');
};

const handleSettings = () => {
  console.log('设置');
};

const handleInfo = () => {
  console.log('信息');
};
</script>`}
>
  <MacWindowExamples.WithToolbar client:load />
</CodeExample>

## 属性

| 属性名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| height | string | 'h-96' | 窗口高度 |
| title | string | '' | 窗口标题 |
| withShadow | boolean | true | 是否显示阴影效果 |
| tabs | array | [] | 标签页字符串数组，如 ["标签1", "标签2", "标签3"] |
| onCloseWindow | function | null | 点击关闭按钮时调用的函数 |
| onMinimizeWindow | function | null | 点击最小化按钮时调用的函数 |
| onMaximizeWindow | function | null | 点击最大化按钮时调用的函数 |
| onTabClick | function | null | 点击标签页时调用的函数，接收标签页值作为参数 |

## 插槽

| 插槽名 | 说明 |
| --- | --- |
| default | 窗口主要内容 |
| sidebar | 侧边栏内容 |
| toolbar | 工具栏内容，位于标题栏右侧 |
| status | 状态栏内容，位于窗口底部 |
