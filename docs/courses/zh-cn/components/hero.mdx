## 基础用法

最简单的Hero区域展示，包含标题、描述和行动按钮

<HeroPackage.HeroContainers.Basic />

## 对齐方式

Hero组件支持内容的左对齐、居中和右对齐

<HeroPackage.HeroContainers.Align />

## 带图片的用法

通过添加图片，使Hero区域更加生动。图片可以设置在不同位置。

<HeroPackage.HeroContainers.WithImage />

## 背景样式

Hero组件支持不同的背景样式，包括纯色和渐变色。

<HeroPackage.HeroContainers.Background />

## 背景图片

添加背景图片可以使Hero区域更加吸引人。还可以添加不同的遮罩效果。

<HeroPackage.HeroContainers.BackgroundImage />

## 带自定义内容的用法

通过插槽添加自定义内容，实现更丰富的展示效果

<HeroPackage.HeroContainers.WithButton />

## 属性说明

| 属性名 | 类型 | 默认值 | 说明 |
|-------|------|-------|------|
| title | string | - | Hero区域的主标题 |
| description | string | - | 标题下方的描述文本 |
| image | Object | - | 可选的图片配置 |
| image.src | string | - | 图片的源地址 |
| image.alt | string | - | 图片的替代文本 |
| links | Array | [] | 链接按钮数组 |
| links[].text | string | - | 链接按钮的文本 |
| links[].href | string | - | 链接按钮的目标地址 |
| background | 'gradient' \| 'plain' | 'gradient' | 背景样式 |
| align | 'center' \| 'left' \| 'right' | 'center' | 内容对齐方式 |
| imagePosition | 'right' \| 'left' \| 'top' \| 'bottom' | 'right' | 图片位置 |
| backgroundImage | string | - | 背景图片的URL |
| backgroundOverlay | 'none' \| 'light' \| 'dark' \| 'primary' \| 'gradient' | 'none' | 背景图片上的遮罩 |
| overlayOpacity | number | 0.7 | 背景遮罩的透明度 |
