## Basic Footer

The simplest way to use a Footer, displaying the site name, slogan, and copyright information

<CodeExample
  code={`<Footer 
  siteName="My Website"
  homeLink="/"
  slogan="Simple and Powerful"
  company="My Company"
  copyright="All Rights Reserved"
/>`}
>
  <Footer 
    siteName="My Website"
    homeLink="/"
    slogan="Simple and Powerful"
    company="My Company"
    copyright="All Rights Reserved"
  />
</CodeExample>

## Footer with Logo

Add a website Logo using the logo property

<CodeExample
  code={`<Footer 
  siteName="My Website"
  homeLink="/"
  slogan="Simple and Powerful"
  company="My Company"
  copyright="All Rights Reserved"
  logo={{ src: getExampleImage({ width: 100, height: 100 }), alt: "Website Logo" }}
/>`}
>
  <Footer 
    siteName="My Website"
    homeLink="/"
    slogan="Simple and Powerful"
    company="My Company"
    copyright="All Rights Reserved"
    logo={{ src: getExampleImage({ width: 100, height: 100 }), alt: "Website Logo" }}
  />
</CodeExample>

## Footer with Social Media Links

Add social media links using the socialLinks property

<CodeExample
  code={`<Footer 
  siteName="My Website"
  homeLink="/"
  slogan="Simple and Powerful"
  company="My Company"
  copyright="All Rights Reserved"
  socialLinks={[
    "https://github.com/myusername",
    "https://twitter.com/myusername",
    "https://linkedin.com/in/myusername"
  ]}
/>`}
>
  <Footer 
    siteName="My Website"
    homeLink="/"
    slogan="Simple and Powerful"
    company="My Company"
    copyright="All Rights Reserved"
    socialLinks={[
      "https://github.com/myusername",
      "https://twitter.com/myusername",
      "https://linkedin.com/in/myusername"
    ]}
  />
</CodeExample>

## Footer with Product Links

Add product links using the products property

<CodeExample
  code={`<Footer 
  siteName="My Website"
  homeLink="/"
  slogan="Simple and Powerful"
  company="My Company"
  copyright="All Rights Reserved"
  products={[
    { name: "Product A", href: "/products/a" },
    { name: "Product B", href: "/products/b" },
    { name: "Product C", href: "/products/c", external: true }
  ]}
/>`}
>
  <Footer 
    siteName="My Website"
    homeLink="/"
    slogan="Simple and Powerful"
    company="My Company"
    copyright="All Rights Reserved"
    products={[
      { name: "Product A", href: "/products/a" },
      { name: "Product B", href: "/products/b" },
      { name: "Product C", href: "/products/c", external: true }
    ]}
  />
</CodeExample>

## Footer with ICP Filing Number

Add an ICP filing number using the icp property (required for Chinese websites)

<CodeExample
  code={`<Footer 
  siteName="My Website"
  homeLink="/"
  slogan="Simple and Powerful"
  company="My Company"
  copyright="All Rights Reserved"
  icp="京ICP备12345678号-1"
/>`}
>
  <Footer 
    siteName="My Website"
    homeLink="/"
    slogan="Simple and Powerful"
    company="My Company"
    copyright="All Rights Reserved"

  />
</CodeExample>
