## 基础用法

最基本的文档布局，包含侧边栏和内容区域

<CodeExample
  code={`<AppLayout
  metaConfig={{
    title: "文档标题",
    description: "文档描述"
  }}
  sidebarConfig={{
    sidebarItems: [
      { title: "入门", items: [
        { href: "/docs/getting-started", text: "快速开始" },
        { href: "/docs/installation", text: "安装" }
      ]}
    ]
  }}
>
  <h1>文档内容</h1>
  <p>这是文档的主要内容</p>
</AppLayout>`}
>
  预览请查看本文档
</CodeExample>

## 无侧边栏模式

通过 showSidebar 属性控制是否显示侧边栏，设置为 false 时不显示侧边栏

<CodeExample
  code={`<AppLayout
  metaConfig={{
    title: "无侧边栏文档",
    description: "这是一个没有侧边栏的文档页面"
  }}
  sidebarConfig={{
    sidebarItems: sidebarItems
  }}
  showSidebar={false}
>
  <h1>文档内容</h1>
  <p>这是文档的主要内容</p>
</AppLayout>`}
>
  预览效果为不显示侧边栏的文档页面
</CodeExample>

## 全宽内容模式

通过设置 `mainContentConfig.fullWidth={true}` 可以让内容区域占满整个宽度，适合创建全宽的展示区域或自定义布局

<CodeExample
  code={`<AppLayout
  metaConfig={{
    title: "全宽内容文档",
    description: "这是一个使用全宽内容区域的文档页面"
  }}
  sidebarConfig={{
    sidebarItems: sidebarItems
  }}
  mainContentConfig={{
    fullWidth: true
  }}
>
  <!-- 全宽Hero部分 -->
  <div class="cosy:bg-primary cosy:text-white cosy:p-10 cosy:text-center">
    <h1 class="cosy:text-3xl">全宽Hero部分</h1>
    <p class="cosy:mt-4">没有容器限制，宽度可以100%占满</p>
  </div>
  
  <!-- 自定义容器部分 -->
  <div class="cosy:container cosy:mx-auto cosy:p-6">
    <p>在全宽模式下，您可以自行控制内容的容器和间距</p>
  </div>
</AppLayout>`}
>
  全宽模式下，内容区域会占满整个可用宽度，适合创建自定义布局
</CodeExample>

## 完整配置示例

AppLayout 组件支持丰富的配置选项，以下是一个完整配置的示例

<CodeExample
  code={`<AppLayout
  metaConfig={{
    title: "完整配置文档",
    description: "这是一个完整配置的文档页面"
  }}
  headerConfig={{
    siteName: "我的文档站",
    logo: myLogo,
    logoHref: "/docs",
    height: "md"
  }}
  sidebarConfig={{
    sidebarItems: sidebarItems
  }}
  showSidebar={true}
  showHeader={true}
  showFooter={true}
  mainContentConfig={{
    showTableOfContents: true,
    fullWidth: false
  }}
  footerConfig={{
    slogan: "构建美好的数字体验"
  }}
  debug={false}
>
  <h1>文档内容</h1>
  <p>这是文档的主要内容</p>
</AppLayout>`}
>
  预览请查看本文档
</CodeExample>

## 主要属性

| 属性名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| metaConfig.title | string | - | 页面标题（必填） |
| metaConfig.description | string | - | 页面描述 |
| headerConfig.siteName | string | "文档中心" | 网站名称 |
| sidebarConfig.sidebarItems | SidebarSection[] | - | 侧边栏项目（必填） |
| showSidebar | boolean | true | 是否显示侧边栏 |
| mainContentConfig.showTableOfContents | boolean | true | 是否显示目录 |
| showHeader | boolean | true | 是否显示页眉 |
| showFooter | boolean | true | 是否显示页脚 |
| headerConfig.height | string | "md" | 页眉高度，可选值：'3xs'/'2xs'/'xs'/'sm'/'md'/'lg'/'xl' |
| mainContentConfig.fullWidth | boolean | false | 内容区域是否占满全宽，设为true时用户可自行控制内容布局 |
| debug | boolean | false | 调试模式，显示各部分边框 |
