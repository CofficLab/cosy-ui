一个用于 Astro 项目的美观实用的组件库，为同一个组织下的多个项目提供统一的 UI 风格。

## 特性

- 🚀 基于 Astro 构建，性能优异
- 📦 组件丰富，满足常见需求
- 📱 响应式设计，完美适配各种设备
- 🔧 TypeScript 支持，开发体验优秀

## 快速开始

通过 npm 安装：

```bash
npm install @coffic/cosy-ui
```

或者使用 pnpm：

```bash
pnpm add @coffic/cosy-ui
```

## 使用示例

在你的 Astro 组件中使用：

```astro
---
import { Button } from '@coffic/cosy-ui';
---

<Button>默认按钮</Button>
```
