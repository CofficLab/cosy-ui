Blog 组件是一个用于展示博客文章的容器组件，提供了标题、副标题、作者信息、日期、封面图片和标签等元素的布局。组件内部使用了响应式设计，确保在不同屏幕尺寸下都有良好的显示效果。

## 基本用法

最简单的用法是只提供`title`属性，并在组件内部放置文章内容：

<CodeExample
  code={`<Blog title="这是一篇博客文章">
  <p>这里是博客文章的内容...</p>
</Blog>`}
>
  <Blog title="这是一篇博客文章">
    <p>这里是博客文章的内容...</p>
  </Blog>
</CodeExample>

## 完整用法

通过添加更多属性，可以创建一个功能完整的博客文章：

<CodeExample
  code={`<Blog
  title="使用 Astro 构建现代网站"
  subtitle="探索 Astro 的强大功能和使用技巧"
  author="张三"
  date={new Date('2023-12-15')}
  cover={{
    src: "https://docs.astro.build/assets/full-logo-light.png",
    alt: "Astro 框架封面图"
  }}
  tags={["Astro", "前端", "教程"]}
>
  <p>这里是博客文章的详细内容...</p>
</Blog>`}
>
  <Blog
    title="使用 Astro 构建现代网站"
    subtitle="探索 Astro 的强大功能和使用技巧"
    author="张三"
    date={new Date('2023-12-15')}
    cover={{
      src: "https://docs.astro.build/assets/full-logo-light.png",
      alt: "Astro 框架封面图"
    }}
    tags={["Astro", "前端", "教程"]}
  >
    <p>这里是博客文章的详细内容...</p>
    <p>Astro 是一个现代的静态站点生成器，它让你可以使用你喜欢的 UI 组件框架（React、Vue、Svelte 等）来构建更快的网站。</p>
  </Blog>
</CodeExample>

## 属性

| 属性名 | 类型 | 默认值 | 说明 |
| ------ | ---- | ------ | ---- |
| title | string | - | 文章标题（必填） |
| subtitle | string | - | 文章副标题 |
| author | string | - | 文章作者 |
| date | Date | - | 发布日期 |
| cover | object | - | 封面图片配置 |
| cover.src | string | - | 封面图片URL |
| cover.alt | string | - | 封面图片替代文本（默认使用标题） |
| tags | string[] | [] | 文章标签数组 |

## 样式结构

组件内部结构和样式说明：

1. **文章容器** - `.article`类提供基本容器，使用`container-type: inline-size`实现容器查询
2. **文章头部** - `.article-header`类包含所有元数据和标题
3. **封面图片** - `.cover-container`和`.cover-image`类处理封面图片的展示
4. **标题区域** - `.header-content`类包含标题、副标题、元数据和标签
5. **元数据** - `.article-meta`类和`.meta-item`类用于作者和日期信息
6. **标签** - `.article-tags`类和`.tag`类用于展示文章标签
7. **内容区域** - `.article-content`类包含主要文章内容

组件内部样式使用了响应式设计原则，在移动端和桌面端有不同的展示效果，并且支持深色模式适配。

## 注意事项

- 所有属性均为可选，除了`title`（必填）
- 当提供`cover`属性时，应确保图片URL有效
- 日期格式化使用`Intl.DateTimeFormat`，默认为中文格式
- 标签样式使用圆角胶囊形状，适合短文本标签
