## 基础用法

最基本的模态对话框，包含标题、内容和关闭按钮

<CodeExample
  code={`<Modal id="basic-modal" title="提示">
  <p>这是一个模态对话框的内容。</p>
  <Button slot="actions" data-modal-target="basic-modal">关闭</Button>
</Modal>

<!-- 触发按钮 -->
<Button data-modal-target="basic-modal">打开模态框</Button>`}
>
  <Modal id="basic-modal" title="提示">
    <p>这是一个模态对话框的内容。</p>
    <Button slot="actions" data-modal-target="basic-modal">关闭</Button>
  </Modal>
  
  <Button data-modal-target="basic-modal">打开模态框</Button>
</CodeExample>

## 确认对话框

包含多个操作按钮的确认对话框

<CodeExample
  code={`<Modal id="confirm-modal" title="确认操作">
  <p>您确定要执行此操作吗？此操作无法撤销。</p>
  <div slot="actions">
    <Button variant="primary">确认</Button>
    <Button variant="ghost" data-modal-target="confirm-modal">取消</Button>
  </div>
</Modal>

<Button variant="warning" data-modal-target="confirm-modal">删除项目</Button>`}
>
  <Modal id="confirm-modal" title="确认操作">
    <p>您确定要执行此操作吗？此操作无法撤销。</p>
    <div slot="actions">
      <Button variant="primary">确认</Button>
      <Button variant="ghost" data-modal-target="confirm-modal">取消</Button>
    </div>
  </Modal>
  
  <Button variant="warning" data-modal-target="confirm-modal">删除项目</Button>
</CodeExample>

## 无标题模态框

不显示标题的简洁模态框，适合图片预览等场景

<CodeExample
  code={`<Modal id="image-modal">
  <div class="cosy:p-4">
    <img src="https://picsum.photos/600/400" alt="图片预览" class="cosy:rounded-md cosy:max-w-full" />
  </div>
</Modal>

<Button variant="ghost" data-modal-target="image-modal">预览图片</Button>`}
>
  <Modal id="image-modal">
    <div class="cosy:p-4">
      <img src="https://picsum.photos/600/400" alt="图片预览" class="cosy:rounded-md cosy:max-w-full" />
    </div>
  </Modal>
  
  <Button variant="ghost" data-modal-target="image-modal">预览图片</Button>
</CodeExample>

## 自定义样式

通过class属性添加自定义样式，实现不同风格的模态框

<CodeExample
  code={`<Modal id="custom-modal" title="自定义模态框" class="cosy:bg-primary cosy:text-primary-content">
  <p>这是一个使用自定义样式的模态框。</p>
  <Button slot="actions" variant="ghost" class="cosy:text-primary-content" data-modal-target="custom-modal">关闭</Button>
</Modal>

<Button variant="primary" data-modal-target="custom-modal">打开自定义模态框</Button>`}
>
  <Modal id="custom-modal" title="自定义模态框" class="cosy:bg-primary cosy:text-primary-content">
    <p>这是一个使用自定义样式的模态框。</p>
    <Button slot="actions" variant="ghost" class="cosy:text-primary-content" data-modal-target="custom-modal">关闭</Button>
  </Modal>
  
  <Button variant="primary" data-modal-target="custom-modal">打开自定义模态框</Button>
</CodeExample>

## 隐藏关闭按钮

通过showCloseButton属性控制是否显示右上角的关闭按钮

<CodeExample
  code={`<Modal id="no-close-button-modal" title="无关闭按钮" showCloseButton={false}>
  <p>此模态框右上角没有关闭按钮，必须使用底部操作区域的按钮关闭。</p>
  <Button slot="actions" data-modal-target="no-close-button-modal">知道了</Button>
</Modal>

<Button data-modal-target="no-close-button-modal">打开模态框</Button>`}
>
  <Modal id="no-close-button-modal" title="无关闭按钮" showCloseButton={false}>
    <p>此模态框右上角没有关闭按钮，必须使用底部操作区域的按钮关闭。</p>
    <Button slot="actions" data-modal-target="no-close-button-modal">知道了</Button>
  </Modal>
  
  <Button data-modal-target="no-close-button-modal">打开模态框</Button>
</CodeExample>
