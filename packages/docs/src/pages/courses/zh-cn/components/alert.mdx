import { Alert, AlertPackage } from '@coffic/cosy-ui';

## 简介

Alert组件用于向用户显示重要的提示信息，支持多种类型(info/success/warning/error)和自定义样式。

组件特性：
- 支持四种预设类型，每种类型有对应的图标和颜色
- 可添加标题使提示更醒目
- 支持自定义class覆盖默认样式
- 响应式设计，适配不同屏幕尺寸

## 基础用法

最简单的提示信息展示：

<AlertPackage.AlertContainers.Basic/>

## 带标题的提示

通过title属性添加标题：

<AlertPackage.AlertContainers.WithTitle/>

## 提示类型

Alert组件支持四种预设类型：

<AlertPackage.AlertContainers.Types/>

## 自定义样式

通过class属性自定义提示框样式：

<AlertPackage.AlertContainers.CustomStyle/>
