Section 组件用于页面内容区块，支持不同的内边距、背景和间距。

## 基础区块

<ComponentPackage.ComponentContainers.Basic />

## 不同背景的区块

通过 background 属性设置区块的背景样式，支持 transparent、white、gray、primary、secondary 和 dark

<ComponentPackage.ComponentContainers.Background />

## 自定义内边距

通过 padding 属性调整区块的内边距大小，支持 none、sm、md、lg 和 xl

<ComponentPackage.ComponentContainers.Padding />

## 内容居中的区块

设置 centered 属性为 true 使区块内容居中显示

<ComponentPackage.ComponentContainers.Centered />

## 不使用容器的区块

设置 container 属性为 false 可以不使用内部的 Container 组件包裹内容

<ComponentPackage.ComponentContainers.Container />

## 自定义容器尺寸

通过 containerSize 属性设置内部容器的尺寸，仅当 container 为 true 时有效

<ComponentPackage.ComponentContainers.ContainerSize />

## 深色背景区块

使用 dark 背景创建深色区块，文字自动变为白色

<ComponentPackage.ComponentContainers.Dark />

展示 Section 组件的多种属性组合使用
