## 简介

Link 组件是一个功能强大的链接组件，不仅支持传统的链接样式，还提供了按钮风格、动画效果和丰富的自定义选项。通过语义化的属性配置，无需手动编写 CSS 类名即可实现各种视觉效果。

**主要特性：**
- 🎨 多种样式变体（default、primary、secondary、text、cta、ghost、light）
- 🔘 完整的按钮风格支持（普通、幽灵、圆形、各种尺寸）
- ✨ 丰富的动画效果（上浮、发光、缩放）
- 📱 响应式设计，支持各种对齐方式
- 🎯 语义化属性，无需手动拼接 CSS 类
- ♿ 完善的无障碍支持

## API 参考

### Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| `href` | 链接地址 | `string` | - |
| `external` | 是否为外部链接，自动新窗口打开 | `boolean` | `false` |
| `variant` | 样式变体 | `'default' \| 'primary' \| 'secondary' \| 'text' \| 'cta' \| 'ghost' \| 'light'` | `'default'` |
| `size` | 尺寸大小 | `'sm' \| 'md' \| 'lg'` | `'md'` |
| `animation` | 动画效果 | `'none' \| 'hover-lift' \| 'hover-glow' \| 'hover-scale'` | `'none'` |
| `block` | 是否为块级显示 | `boolean` | `false` |
| `btn` | 是否启用按钮风格 | `boolean` | `false` |
| `circle` | 是否为圆形按钮（需配合 `btn` 使用） | `boolean` | `false` |
| `ghost` | 是否为幽灵按钮（需配合 `btn` 使用） | `boolean` | `false` |
| `color` | 按钮颜色（需配合 `btn` 使用） | `'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | - |
| `align` | 对齐方式 | `'left' \| 'center' \| 'right'` | - |
| `rounded` | 是否添加圆角 | `boolean` | `false` |
| `noUnderline` | 是否移除下划线 | `boolean` | `false` |
| `fullWidth` | 是否占满宽度 | `boolean` | `false` |
| `centerText` | 是否居中文本（兼容属性） | `boolean` | `false` |
| `debug` | 是否显示调试边框 | `boolean` | `false` |
| `class` | 自定义 CSS 类名 | `string` | - |
| `...rest` | 其他 HTML `<a>` 标签属性 | `HTMLAttributes<'a'>` | - |

### 使用建议

1. **语义化优先**：优先使用 `variant` 属性而不是自定义样式
2. **按钮风格**：需要按钮样式时使用 `btn` 属性，配合 `color` 属性设置颜色
3. **动画适度**：动画效果应适度使用，避免过度干扰用户
4. **无障碍支持**：外部链接会自动添加 `rel="noopener noreferrer"` 属性
5. **响应式设计**：使用 `block` 和 `align` 属性实现响应式布局

### 最佳实践

```astro
<!-- ✅ 推荐：使用语义化属性 -->
<Link href="/signup" btn color="primary" size="lg">注册账号</Link>

<!-- ❌ 不推荐：手动拼接 CSS 类 -->
<Link href="/signup" class="cosy:btn cosy:btn-primary cosy:btn-lg">注册账号</Link>

<!-- ✅ 推荐：外部链接使用 external 属性 -->
<Link href="https://example.com" external>外部链接</Link>

<!-- ✅ 推荐：组合使用多个属性 -->
<Link href="/cta" btn ghost color="accent" animation="hover-scale">
  立即体验
</Link>
```
