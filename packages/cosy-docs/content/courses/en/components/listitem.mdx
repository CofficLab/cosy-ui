ListItem is a list item component that supports a loading progress bar animation, suitable for scenarios requiring async operation feedback.

## Introduction

The ListItem component displays list item content and shows a left-to-right progress bar animation during async operations, improving user feedback experience.

## Props

| Name     | Type    | Default | Description                    |
|----------|---------|---------|--------------------------------|
| loading  | boolean | false   | Whether to show the progress bar |
| duration | number  | 1500    | Progress bar animation duration (ms) |
