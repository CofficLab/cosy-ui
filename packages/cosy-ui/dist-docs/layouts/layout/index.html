布局组件

这些组件可以帮助你快速构建页面布局

Container 容器

Container组件用于限制内容宽度并居中显示，是最基础的布局组件。

小尺寸容器 (sm)

中尺寸容器 (md) - 默认

大尺寸容器 (lg)

使用示例

<Container>
  <p>内容将被限制在一个合理的宽度内并居中显示</p>
</Container>

<Container size="sm" padding="lg">
  <p>小尺寸容器，大内边距</p>
</Container>

Grid 网格

Grid组件提供了一个灵活的网格布局系统，支持响应式列数和间距。

2列网格

第二列

3列网格

第一列

响应式网格 (移动端1列，平板2列，桌面3列)

第一列
第二列
第三列

使用示例

<Grid cols=3 gap="md">
  <div>第一列</div>
  <div>第二列</div>
  <div>第三列</div>
</Grid>

<Grid cols=[object Object] gap="lg">
  <div>响应式网格</div>
  <div>在不同屏幕尺寸下</div>
  <div>显示不同的列数</div>
</Grid>

Flex 弹性布局

Flex组件提供了一个灵活的弹性布局系统，支持不同的方向、对齐和间距。

水平居中对齐 (默认)

项目1
项目2
项目3

垂直排列，左对齐

项目1
项目2
项目3

水平两端对齐

左侧
中间
右侧

使用示例

<Flex>
  <div>默认水平排列</div>
  <div>居中对齐</div>
</Flex>

<Flex direction="column" align="start" justify="between" gap="lg">
  <div>垂直排列</div>
  <div>左对齐</div>
  <div>两端对齐</div>
  <div>大间距</div>
</Flex>

Section 区块

Section组件用于创建页面中的内容区块，支持不同的内边距、背景和间距。

基础区块

区块标题

区块内容将被包裹在一个合适的容器中

带背景色的区块

居中区块

带有背景色的居中内容区块

使用示例

<Section>
  <h2>默认区块</h2>
  <p>内容将被包裹在一个合适的区块中</p>
</Section>

<Section padding="lg" background="gray" centered=true>
  <h2>自定义区块</h2>
  <p>大内边距，灰色背景，内容居中</p>
</Section>

  <div>第一个元素</div>
  <div>第二个元素</div>
  <div>第三个元素</div>
</Stack>

<Stack gap="lg" dividers=true>
  <div>带分隔线的堆叠</div>
  <div>大间距</div>
  <div>自动垂直排列</div>
</Stack>

组合使用布局组件

这些布局组件可以组合使用，创建复杂的页面布局。

组合示例

<Section padding="lg" background="gray">
  <Container size="lg">
    <Stack gap="lg">
      <h2>页面标题</h2>
      
      <Grid cols=[object Object]>
        <Stack>
          <h3>左侧内容</h3>
          <p>一些描述文字...</p>
        </Stack>
        
        <Flex direction="column" align="center">
          <img src="..." alt="示例图片" />
          <p>图片说明</p>
        </Flex>
      </Grid>
    </Stack>
  </Container>
</Section>
