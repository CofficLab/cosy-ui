Container 组件示例

基础用法

最简单的容器用法，默认使用中等尺寸和中等内边距

内容将被限制在一个合理的宽度内并居中显示 
<Container>
  <div class="bg-gray-100 p-4 text-center">
    内容将被限制在一个合理的宽度内并居中显示
  </div>
</Container>

不同尺寸

Container 组件支持六种不同的尺寸：xs、sm、md、lg、xl 和 full

超小尺寸容器
小尺寸容器
中等尺寸容器（默认）
大尺寸容器

全宽容器
<Container size="xs">
  <div class="bg-gray-100 p-4 text-center">超小尺寸容器</div>
</Container>

  <div class="bg-gray-100 p-4 text-center">小尺寸容器</div>
</Container>

<Container size="md">
  <div class="bg-gray-100 p-4 text-center">中等尺寸容器（默认）</div>
</Container>

<Container size="lg">
  <div class="bg-gray-100 p-4 text-center">大尺寸容器</div>
</Container>

  <div class="bg-gray-100 p-4 text-center">超大尺寸容器</div>

  <div class="bg-gray-100 p-4 text-center">全宽容器</div>
</Container>

Container 组件支持五种不同的内边距：none、sm、md、lg 和 xl

无内边距
小内边距
中等内边距（默认）
大内边距
超大内边距

居中选项

通过 centered 属性控制容器是否居中显示

居中显示（默认）
不居中显示
<Container centered={true}>
  <div class="bg-gray-100 p-4 text-center">居中显示（默认）</div>
</Container>

<Container centered={false}>
  <div class="bg-gray-100 p-4 text-center">不居中显示</div>
</Container>

自定义样式

可以通过 class 属性添加自定义样式

带有自定义样式的容器 
<Container class="bg-blue-50 rounded-lg shadow-md">
  <div class="p-4 text-center">
    带有自定义样式的容器
  </div>
</Container>

实际应用场景

在实际应用中的常见用法

页面标题

侧边栏内容

主要内容区域，使用容器组件可以确保内容在各种屏幕尺寸下都有合适的宽度和边距。

容器组件是布局的基础，通常用于页面的最外层或主要内容区域。

<Container size="lg">
  <header class="py-4 border-b mb-6">
    <h2 class="text-xl font-bold">页面标题</h2>
  </header>
  
  <main class="grid grid-cols-1 md:grid-cols-3 gap-6">
    <aside class="md:col-span-1 bg-gray-50 p-4 rounded">
      侧边栏内容
    </aside>
    
    <section class="md:col-span-2">
      <p class="mb-4">主要内容区域，使用容器组件可以确保内容在各种屏幕尺寸下都有合适的宽度和边距。</p>
      <p>容器组件是布局的基础，通常用于页面的最外层或主要内容区域。</p>
    </section>
  </main>
</Container>
