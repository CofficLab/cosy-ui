Article 文章

用于展示文章内容的组件，支持标题、副标题、作者信息、发布日期、封面图片和标签等元素。

引言

这是一个示例文章，展示了 Article 组件的各种功能和样式。文章组件支持完整的 Markdown 样式，包括但不限于：

  • 标题层级
  • 段落文本
  • 列表（有序和无序）
  • 代码块
  • 引用
  • 表格

组件 API

Props

属性 类型 默认值 描述
title string - 文章标题
subtitle string? undefined 文章副标题
author string? undefined 作者名称
date Date? undefined 发布日期
cover { src: string; alt?: string; }? undefined 封面图片
tags string[] [] 文章标签

Slots

默认插槽用于放置文章的主要内容。支持 HTML 和 Markdown 格式的内容。
