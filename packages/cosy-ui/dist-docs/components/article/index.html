基础用法

</Article>

引用和代码

支持引用块、行内代码和代码块

引用块

这是一段引用内容，可以用来强调重要信息或引用他人的话。 引用块使用左侧边框和斜体样式，使其在视觉上与普通文本区分开来。

行内代码

可以在段落中使用 行内代码 来标记代码片段或命令。

代码块

// 这是一个代码块
function example() {
  return "示例代码";
}
<Article>
  <h3>引用块</h3>
  <blockquote>
    这是一段引用内容，可以用来强调重要信息或引用他人的话。
    引用块使用左侧边框和斜体样式，使其在视觉上与普通文本区分开来。
  </blockquote>
  
  <h3>行内代码</h3>
  <p>可以在段落中使用 <code>行内代码</code> 来标记代码片段或命令。</p>
  
  <h3>代码块</h3>
  <pre><code>// 这是一个代码块
function example() {
  return "示例代码";
}</code></pre>
</Article>
