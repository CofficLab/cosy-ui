前端开发工程师

/>

用法示例

<TeamMembers
  members={[
    {
      name: "张三",
      role: "前端开发工程师",
      avatar: "https://i.pravatar.cc/300?img=1",
      bio: "专注于 React 和 TypeScript 开发，热爱开源社区，喜欢分享技术经验。",
      socialLinks: [
        { platform: 'github', url: 'https://github.com' },
        { platform: 'twitter', url: 'https://twitter.com' }
      ]
    },
    // 更多团队成员...
  ]}
  columns={3} // 可选，默认为3
/>

属性说明

TeamMember 属性

属性名 类型 默认值 说明
name string - 团队成员姓名
role string - 团队成员职位
avatar ImageMetadata | string - 头像图片（可以是URL字符串或Astro图片对象）
bio string - 个人简介
socialLinks SocialLink[] [] 社交媒体链接数组
class string '' 自定义CSS类名

TeamMembers 属性

属性名 类型 默认值 说明
members TeamMemberData[] - 团队成员数据数组
columns 2 | 3 | 4 3 显示的列数
class string '' 自定义CSS类名
