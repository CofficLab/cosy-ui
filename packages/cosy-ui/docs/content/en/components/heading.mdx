# Heading

The Heading component is used to create headings of various levels, providing consistent typography styles and flexible customization options. This component is ideal for content pages that require a clear hierarchical structure, such as documentation, blogs, and product introductions.

## Basic Usage

Use the `level` property to specify the heading level (1-6):

<CodeExample
  code={`<Heading level={1}>Heading Level 1</Heading>
<Heading level={2}>Heading Level 2</Heading>
<Heading level={3}>Heading Level 3</Heading>
<Heading level={4}>Heading Level 4</Heading>
<Heading level={5}>Heading Level 5</Heading>
<Heading level={6}>Heading Level 6</Heading>`}
>
  <Heading level={1}>Heading Level 1</Heading>
  <Heading level={2}>Heading Level 2</Heading>
  <Heading level={3}>Heading Level 3</Heading>
  <Heading level={4}>Heading Level 4</Heading>
  <Heading level={5}>Heading Level 5</Heading>
  <Heading level={6}>Heading Level 6</Heading>
</CodeExample>

## Custom Styles

You can customize heading styles using the `color` and `underline` properties:

<CodeExample
  code={`<Heading level={2} color="primary">Primary Color Heading</Heading>
<Heading level={2} color="secondary">Secondary Color Heading</Heading>
<Heading level={2} color="accent">Accent Color Heading</Heading>
<Heading level={2} color="muted">Muted Color Heading</Heading>
<Heading level={2} underline>Underlined Heading</Heading>`}
>
  <Heading level={2} color="primary">Primary Color Heading</Heading>
  <Heading level={2} color="secondary">Secondary Color Heading</Heading>
  <Heading level={2} color="accent">Accent Color Heading</Heading>
  <Heading level={2} color="muted">Muted Color Heading</Heading>
  <Heading level={2} underline>Underlined Heading</Heading>
</CodeExample>

## Alignment

Control heading alignment using the `align` property:

<CodeExample
  code={`<Heading level={3} align="left">Left Aligned Heading</Heading>
<Heading level={3} align="center">Center Aligned Heading</Heading>
<Heading level={3} align="right">Right Aligned Heading</Heading>`}
>
  <Heading level={3} align="left">Left Aligned Heading</Heading>
  <Heading level={3} align="center">Center Aligned Heading</Heading>
  <Heading level={3} align="right">Right Aligned Heading</Heading>
</CodeExample>

## Headings with Anchors

Create headings with anchor links using the `id` and `anchor` properties:

<CodeExample
  code={`<Heading level={2} id="section-example" anchor>Heading with Anchor</Heading>`}
>
  <Heading level={2} id="section-example" anchor>Heading with Anchor</Heading>
</CodeExample>

When users hover over the heading, the anchor icon will appear, which can be clicked to jump to the corresponding section.

## Properties

| Property | Type | Default | Description |
| -------- | ---- | ------- | ----------- |
| level | number | 2 | Heading level, corresponding to h1-h6 tags |
| id | string | - | Heading ID, used for anchor links |
| anchor | boolean | false | Whether to display the anchor link icon |
| underline | boolean | false | Whether to display an underline |
| align | string | 'left' | Text alignment: 'left', 'center', 'right' |
| color | string | 'default' | Heading color: 'default', 'primary', 'secondary', 'accent', 'muted' |
| class | string | - | Custom class name |

## Best Practices

- Follow heading hierarchy, don't skip levels (e.g., using h3 directly after h1)
- Use the anchor property to create linkable anchors for important sections in long documents
- For important content or sections that need emphasis, use the `color` property to set different colors
- Maintain consistent heading styles throughout a page to improve readability and user experience
