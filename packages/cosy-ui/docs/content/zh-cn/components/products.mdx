## 属性

| 属性名 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| products | ProductItem[] | - | 产品数据数组，每项包含ProductCard组件的所有属性（除class外） |
| layout | 'grid' \| 'list' | 'grid' | 布局类型，网格或列表 |
| columns | number \| ResponsiveColumns | \{ base: 1, sm: 2, md: 3, lg: 4 } | 列数配置，可以是固定数字或响应式对象 |
| cardSize | 'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' | 'md' | 卡片尺寸，应用于所有ProductCard |
| gap | 'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' | 'md' | 卡片间距 |
| equalHeight | boolean | true | 是否启用等高卡片 |
| descriptionLines | number | - | 描述文本的最大行数 |
| class | string | - | 自定义类名 |
