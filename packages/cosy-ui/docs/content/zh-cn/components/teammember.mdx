## 单个团队成员展示

展示单个团队成员的信息卡片

<CodeExample
  code={`<TeamMember
  name="张三"
  role="前端开发工程师"
  avatar={getAvatarImage({ width: 300, height: 300 })}
  bio="拥有5年前端开发经验，专注于React和Vue生态系统。热爱开源项目和新技术探索。"
/>`}
>
  <TeamMember
    name="张三"
    role="前端开发工程师"
    avatar={getAvatarImage({ width: 300, height: 300 })}
    bio="拥有5年前端开发经验，专注于React和Vue生态系统。热爱开源项目和新技术探索。"
  />
</CodeExample>
