## 基础用法

最基本的文档布局，包含侧边栏和内容区域

<CodeExample
  code={`<DocumentationLayout
  title="文档标题"
  description="文档描述"
  sidebarItems={[
    { title: "入门", items: [
      { href: "/docs/getting-started", text: "快速开始" },
      { href: "/docs/installation", text: "安装" }
    ]}
  ]}
>
  <h1>文档内容</h1>
  <p>这是文档的主要内容</p>
</DocumentationLayout>`}
>
  预览请查看本文档
</CodeExample>

## 无侧边栏模式

通过 showSidebar 属性控制是否显示侧边栏，设置为 false 时不显示侧边栏

<CodeExample
  code={`<DocumentationLayout
  title="无侧边栏文档"
  description="这是一个没有侧边栏的文档页面"
  sidebarItems={sidebarItems}
  showSidebar={false}
>
  <h1>文档内容</h1>
  <p>这是文档的主要内容</p>
</DocumentationLayout>`}
>
  预览效果为不显示侧边栏的文档页面
</CodeExample>

## 完整配置示例

DocumentationLayout 组件支持丰富的配置选项，以下是一个完整配置的示例

<CodeExample
  code={`<DocumentationLayout
  title="完整配置文档"
  description="这是一个完整配置的文档页面"
  siteName="我的文档站"
  logo={myLogo}
  logoHref="/docs"
  sidebarItems={sidebarItems}
  showSidebar={true}
  showTableOfContents={true}
  showHeader={true}
  showFooter={true}
  debug={false}
  headerHeight="md"
  footerSlogan="构建美好的数字体验"
>
  <h1>文档内容</h1>
  <p>这是文档的主要内容</p>
</DocumentationLayout>`}
>
  预览请查看本文档
</CodeExample>

## 主要属性

| 属性名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| title | string | - | 页面标题（必填） |
| description | string | - | 页面描述 |
| siteName | string | "文档中心" | 网站名称 |
| sidebarItems | SidebarSection[] | - | 侧边栏项目（必填） |
| showSidebar | boolean | true | 是否显示侧边栏 |
| showTableOfContents | boolean | true | 是否显示目录 |
| showHeader | boolean | true | 是否显示页眉 |
| showFooter | boolean | true | 是否显示页脚 |
| headerHeight | string | "md" | 页眉高度，可选值：'3xs'/'2xs'/'xs'/'sm'/'md'/'lg'/'xl' |
| debug | boolean | false | 调试模式，显示各部分边框 |
