## 基础页脚

最简单的 Footer 使用方式，展示网站名称、标语和版权信息

<CodeExample
  code={`<Footer 
  siteName="我的网站"
  homeLink="/"
  slogan="简单而强大"
  company="我的公司"
  copyright="保留所有权利"
/>`}
>
  <Footer 
    siteName="我的网站"
    homeLink="/"
    slogan="简单而强大"
    company="我的公司"
    copyright="保留所有权利"
  />
</CodeExample>

## 带有 Logo 的页脚

<CodeExample

## 带有社交媒体链接的页脚

通过 socialLinks 属性添加社交媒体链接

<CodeExample
  code={`<Footer 
  siteName="我的网站"
  homeLink="/"
  slogan="简单而强大"
  company="我的公司"
  copyright="保留所有权利"
  socialLinks={[
    "https://github.com/myusername",
    "https://twitter.com/myusername",
    "https://linkedin.com/in/myusername"
  ]}
/>`}
>
  <Footer 
    siteName="我的网站"
    homeLink="/"
    slogan="简单而强大"
    company="我的公司"
    copyright="保留所有权利"
    socialLinks={[
      "https://github.com/myusername",
      "https://twitter.com/myusername",
      "https://linkedin.com/in/myusername"
    ]}
  />
</CodeExample>

## 带有产品链接的页脚

通过 products 属性添加产品链接

<CodeExample
  code={`<Footer 
  siteName="我的网站"
  homeLink="/"
  slogan="简单而强大"
  company="我的公司"
  copyright="保留所有权利"
  products={[
    { name: "产品A", href: "/products/a" },
    { name: "产品B", href: "/products/b" },
    { name: "产品C", href: "/products/c", external: true }
  ]}
/>`}
>

      { name: "产品B", href: "/products/b" },

    ]}
  />
</CodeExample>
