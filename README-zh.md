这是一个 Astro 组件库，为同一个组织下的多个项目提供统一的 UI 风格。

## 快速开始

Installation

```bash
npm install @coffic/cosy-ui
```

Usage in an Astro component:

```js
---
import { Button } from "@coffic/cosy-ui";
---
<Button>Hi</Button>
```

## 开发

### 安装依赖

```bash
pnpm install
```

### 启动

```bash
pnpm dev
```

### 构建

```bash
pnpm build
```
